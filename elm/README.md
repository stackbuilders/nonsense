# Nonsense!

> a contestant shrugged the dictionary

Generate nonsense English sentences in [Elm][elm] using [Wordbot][wordbot].

To start the application, run:

```
npm install
npm run elm -- reactor
```

And open http://localhost:8000/src/Nonsense.elm.

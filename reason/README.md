# Nonsense!

> a parsnip scratched a bottle of nail polish

Generate nonsense English sentences in [Reason][reason] using
[Wordbot][wordbot].

To start the application, run:

```
npm install
npm run start
# in another tab
npm run serve
```

And open http://localhost:8000.

## Other versions

* [Elm](/elm/)

[reason]: https://reasonml.github.io
